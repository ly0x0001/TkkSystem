## Description

- TkkSystem是为嘉庚学子开发的一款校园App。
- 对Web教务系统登陆接口进行抓包分析
- 模拟实现超级课程表的课表显示方式
- 对日常高频功能进行实行

## Features

- 教务系统登录
- 课程表的获取，可切换不同学期的课程表
- 考试安排的获取
- 考试成绩的获取
